## Version History

### Version Numbering
- **Major** (X.0.0): New game features, major system changes
- **Minor** (0.X.0): New content, feature additions
- **Patch** (0.0.X): Bug fixes, balance changes, small improvements

### Release Schedule
- **Major releases**: Every 6 months
